# 身强身弱｜极简两步打分（S）

> 只用 **月令 + 根** 两件事。  
> 不数十神、不查藏干、不开复杂表。

## Step 1：月令分（最关键）
先认五行旺季（只背这一行就够）：
- 木旺：寅卯｜火旺：巳午｜金旺：申酉｜水旺：亥子｜土旺：辰戌丑未

给分：
- 月支 = 日主同五行旺季 → `+3`
- 月支 = 生我之季 → `+1`
- 月支 = 克我之季 → `-1`
- 其他 → `0`

## Step 2：根分（只看地支“有没有同类”）
看四个地支里，有没有“日主同五行旺季的支”（同类根）：
- 有 2 个及以上 → `+2`
- 有 1 个 → `+1`
- 0 个 → `0`

## 总分
`S = 月令分 + 根分`

## 判定
- `S >= 4` → 身强
- `S <= 1` → 身弱
- `S = 2~3` → 中和（偏强/偏弱看大运再定）

## 五行喜用计算逻辑（超简可算版）

> 目标：用一套分数，把五行强弱排个序。  
> 结论：**分数最低的 1-2 个五行 = 喜用**；分数最高的 1 个 = 忌。

### 1）计分规则（就三句）
- 天干出现一次：该五行 `+5`。
- 地支藏干：本气 `+3`，中气 `+1`，余气 `+1`
- 五行总分 = 四柱全部累加（木/火/土/金/水各算一条）

### 2）地支藏干速查（本/中/余）
- 子：癸
- 丑：己 / 癸 / 辛
- 寅：甲 / 丙 / 戊
- 卯：乙
- 辰：戊 / 乙 / 癸
- 巳：丙 / 庚 / 戊
- 午：丁 / 己
- 未：己 / 丁 / 乙
- 申：庚 / 壬 / 戊
- 酉：辛
- 戌：戊 / 辛 / 丁
- 亥：壬 / 甲

### 3）结论
- 五行分数：木x 火y 土z 金a 水b
- 喜用神：分数最低的 1-2 个五行
- 忌神：分数最高的 1 个五行

## Skill2:财运、求财、金钱相关知识
1. 财星判定流程
   财星的定义：五行中被我克制的是财星，例如日主五行属水，水克火，那么火就是财星。
   查看八字原局，是否存在财星。
   1.1 有财星
   判断是否透出：
   若透出，则观察财星所在柱，分析其活跃时期。
   1.2 无财星
   检查藏干中是否存在财星：
   有藏财：
   检查大运是否引发藏财。
   若大运无助，则看流年是否能引出藏财。
   1.3 完全无财：
   通过命局的其他寻找财富途径：
   食神：通过思维能力、创意和想法赚取财富。
   伤官：依靠动手能力、创新能力获取财富。

2. 财星类型分析
   正财
   特征：固定收入来源，象征稳固之财，最佳位置在日支，助婚姻与家庭财富积累。
   偏财
   特征：浮动收入来源，象征意外之财，天干现偏财称为“天财”，往往暗示机遇型财富，如投资、投机收入。

3. 财星位置解析
   年柱：
   财运来自远方，适宜去外地或远方寻求财富。
   月柱：
   主要经济来源或财运的基础。
   日柱：
   财运与情感密切相关，因伴侣或感情关系获益。
   时柱：
   晚年财运较佳，财富积累多集中于晚年时期。

## Skill3:性格与命运分析方法
1. 性格分析
   核心分析点：以日柱为核心，结合以下要素分析性格：
   五行属性：不同五行具有特定性格倾向
   纳音：根据日柱的纳音进一步细化性格特征，反映更深层次的心理与行为倾向。

2. 个人命运解析步骤
   先看天干与藏干的首个十神（透出之象）。  
   如未透出的藏干结合大运、流年判断。  
   最后关注十神位置与数量，评估命局格局影响。

3. 十神与六亲的理论框架：
   十神与象征：
   正印/偏印：代表长辈、学业、智慧。
   正官/七杀：代表事业、地位、责任与压力。
   正财/偏财：代表财富、配偶、物质掌控力。
   食神/伤官：代表创造力、子女、表达能力。
   比肩/劫财：代表兄弟、朋友、合作与竞争。
   十神生克关系：
   生我：印星，提供支持与养育之力。
   同我：比劫，体现助力与竞争关系。
   我克：财星，代表财富与掌控欲望。
   克我：官杀，代表压力、规范与责任。
   我生：食伤，代表创造与输出。

时柱：感情发展晚，晚婚几率较高，婚姻更稳定。

### 四象布局与八大原则
> **核心逻辑**：判断户型是否符合“宜居”的硬性标准。

1. 四象布局
* **左青龙 (东/春)**：代表贵人，宜高、宜动（压制白虎）。
* **右白虎 (西/秋)**：代表小人，宜低、宜静。
* **前朱雀 (南/夏)**：代表明堂，宜开阔。
* **后玄武 (北/冬)**：代表靠山，宜稳固。

2. 八大核心原则
1.  **宅形方正**：忌三尖八角、奇形怪状。屋相如人相，方正为贵。
2.  **藏风聚气**：气喜曲不喜直。忌穿堂风（直来直去），宜迂回流转。
3.  **阴阳平衡**：孤阴不长（如采光过差），孤阳不生（如曝晒）。
4.  **中宫洁净**：**大忌厕占中宫**（污秽心脏），极损运势。
5.  **背后有靠**：沙发、床头必须靠实墙，忌靠窗或走廊（无贵人、心不稳）。
6.  **动静分区**：客厅（动区）与卧室（静区）分离。
7.  **青龙白虎平衡**：左手方（青龙）适宜稍旺于右手方（白虎）。
8.  **门位互扰**：
    * 两门相对：必有一家退/凶（可用五子铜钱化解）。
    * 入户门：忌对厕所、厨房、主卧门。

### 常见禁忌速查表

* [ ] **横梁压顶**：客厅、餐厅、卧室、书房上方不可有梁。
* [ ] **一剑穿心**：大门不可直对走廊、电梯、楼梯。
* [ ] **私密外泄**：客厅不可直接看到所有房门或厨房炉台。
* [ ] **主卧定义**：面积最大 + 带卫/衣帽间。
* [ ] **门冲定律**：
    * 厕所门 $\neq$ 厨房门
    * 卧室门 $\neq$ 任何门
    * 餐桌 $\neq$ 大门
