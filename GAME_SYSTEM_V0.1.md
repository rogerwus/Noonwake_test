# 人生副本 / 玄学融合怪
## Game System v0.1

> 目标：不是做“算命展示页”，而是做一个 **抽卡 + 装备 + 刷怪闯关 + 排行榜** 的微信小游戏，玄学命理只负责提供差异化 build、成长解释和分榜机制。

---

## 1. 产品一句话

**用生辰生成你的命格职业，然后抽命器、刷秘境、打流年 Boss、冲同命理排行榜。**

---

## 2. 产品定位

### 不做什么
- 不做八字报告工具
- 不做静态 RPG 面板
- 不做纯剧情选择游戏
- 不做一局结束就没内容的网页 demo

### 要做什么
- 微信小游戏式数值成长
- 玄学皮下的成熟商业玩法
- 低门槛上手，高复玩，强排行，强养成
- 玄学不是包装，而是 **玩法输入层**

---

## 3. 核心玩法结构

### 核心三环

#### A. 外循环：抽卡 / 养成 / 配装 / 冲榜
- 抽命器
- 抽神通
- 抽守护灵
- 强化装备
- 洗炼词条
- 调整 build
- 冲总榜 / 分榜

#### B. 中循环：刷关 / 打 Boss / 掉装备 / 升层
- 进入秘境
- 自动或半自动战斗
- 小怪掉材料
- Boss 掉高品质装备/稀有材料
- 推更高层数

#### C. 内循环：战斗中的触发与克制
- 普攻 / 技能 / 被动触发
- 装备词条联动
- 命格适配加成
- 五行克制
- 运势浮动影响掉率/暴击/触发率

---

## 4. 玄学接入点

玄学必须进入系统，不是只做 UI 文案。

### 4.1 命格决定开局职业
| 日主五行 | 职业定位 | 玩法倾向 |
|---|---|---|
| 木 | 青龙游侠 | 成长、召唤、恢复 |
| 火 | 朱雀战法 | 爆发、AOE、连击 |
| 土 | 麒麟守护 | 肉盾、反伤、稳定推进 |
| 金 | 白虎刺客 | 暴击、穿透、速杀 |
| 水 | 玄武谋士 | 控制、谋略、技能流 |

### 4.2 身强身弱决定承压模型
- **身强**：更适合高风险高收益 build
- **身弱**：更吃装备适配和资源管理
- **中和**：更容易走均衡流

### 4.3 喜用神决定推荐路线
- 喜金：暴击、穿透、精密装
- 喜木：成长、恢复、持续收益
- 喜水：控制、技能触发、智力装
- 喜火：爆发、速度、连击装
- 喜土：护盾、反伤、生存装

### 4.4 忌神决定风险来源
- 忌神装备穿上会有反噬
- 忌神 Boss/地图会提高难度
- 忌神日的副本波动更大

### 4.5 命理决定排行榜分层
- 全服总榜
- 同日主榜
- 同日柱榜
- 同命格职业榜
- 同年龄段榜
- 同地区榜
- 今日运势榜

## 5. 抽卡系统

不抽普通角色，抽玄学相关内容。

### 5.1 三大池子

#### 命器池（装备）
例子：
- 青龙玉佩
- 白虎断刃
- 玄武镜
- 朱雀火羽
- 麒麟镇印

#### 神通池（技能）
例子：
- 七杀爆发
- 正印护体
- 偏财暴击
- 食神续航
- 伤官破甲

#### 守护灵池（宠物/跟班）
例子：
- 招财狐
- 镇煞乌鸦
- 山海异兽
- 文昌书灵
- 业火灵

### 5.2 稀有度
- R
- SR
- SSR
- UR

### 5.3 抽卡体验目标
- 3 秒内爽点
- 强烈出货反馈
- 抽到后立即可投入 build
- 新手前几抽必须能组成一套基础流派

## 6. 装备系统

### 6.1 装备位（MVP）
- 武器
- 护符
- 防具
- 饰品
- 命盘核心

### 6.2 装备属性
- 基础属性：ATK / DEF / HP / INT / CHA / LUK
- 五行属性：木火土金水
- 稀有度
- 命格适配度
- 特殊词条

### 6.3 词条示例
- 暴击率 +8%
- 技能触发率 +12%
- 控场时间 +10%
- 对火系 Boss 伤害 +15%
- 喜水命格额外增益 +20%
- 忌土时受到反噬 -5% HP/关

### 6.4 套装效果
- 青龙套：成长型收益递增
- 白虎套：暴击斩杀
- 玄武套：技能回复/控制
- 朱雀套：AOE 爆发
- 麒麟套：反伤与减伤

## 7. 刷怪闯关系统

### 7.1 地图包装
普通副本名称全部玄学化：
- 命宫试炼
- 五行秘境
- 十神塔
- 大运裂隙
- 流年劫关
- 太岁降临

### 7.2 怪物包装
- 劫财贼
- 枭印妖
- 偏财魔
- 七杀影
- 太岁分身
- 五行守卫

### 7.3 战斗形式（MVP）
- 轻量自动战斗
- 玩家主要决策放在配装、技能、抽卡、上阵守护灵
- 战斗中保留少量主动技能释放/大招时机

### 7.4 Boss 设计（首批）
- 太岁冲命
- 破财煞
- 桃花劫
- 七杀压顶
- 枭神夺食

每个 Boss 绑定：
- 五行倾向
- 克制/被克制关系
- 特定命格难度修正
- 特定套装优势

## 8. 成长系统

### 8.1 玩家成长目标
- 提升战力
- 优化 build
- 凑套装
- 解锁更高层秘境
- 冲分榜

### 8.2 成长维度
- 升级装备
- 强化神通
- 守护灵突破
- 命盘节点解锁
- 洗炼词条
- 收集稀有套装

### 8.3 命盘成长
命盘做成一个中长期成长树：
- 增加初始掉率
- 增加特定五行伤害
- 增加特定 Boss 减伤
- 增加同命理榜加成称号

## 9. 排行榜系统

排行榜是传播核心。

### 9.1 必做榜单
1. 全服总战力榜
2. 全服爬塔榜
3. 同日主榜
4. 同日柱榜
5. 同职业榜
6. 今日 Boss 伤害榜

### 9.2 最有传播性的榜单
- **同命理榜**
- **同日柱榜**
- **同城同命格榜**

### 9.3 排行奖励
- 称号
- 边框
- 稀有抽卡券
- 限时命器皮肤

---

## 10. 每日留存系统

### 10.1 每日运势
每天刷新：
- 今日幸运五行
- 今日忌神预警
- 今日地图掉率加成
- 今日 Boss 特殊加成

### 10.2 每日任务
- 刷 3 次秘境
- 强化 1 次装备
- 抽 1 次命器
- 打 1 次 Boss
- 完成 1 次榜单挑战

### 10.3 周期活动
- 太岁周赛
- 同命理冲榜周
- 五行轮换赛季

## 11. 变现结构

### 11.1 主要付费点
- 抽卡
- 十连折扣
- 每日礼包
- 月卡
- 强化材料包
- 洗炼材料包
- 守护灵皮肤
- 称号/边框

### 11.2 控制原则
- 免费用户也能爽刷和冲基础榜
- 付费用户更快成型、更快拿稀有 build
- 不做纯 Pay-to-Win，优先做 Pay-for-Speed / Pay-for-Variety

---

## 12. MVP 范围

### 第一版必须有
1. 生辰生成命格职业
2. 基础战斗关卡（至少 3 张图）
3. 命器池 + 神通池
4. 基础装备掉落与强化
5. 轻量自动战斗
6. 1 个 Boss 地图
7. 总榜 + 同日主榜

### 第一版不要做
- 复杂剧情
- 实时 PvP
- 社交聊天
- 太多职业分支
- 太多地图层数
- 太重的 meta 经济

## 14. 成功标准

### 玩家感知
- 3 秒懂：这是玄学刷宝游戏
- 30 秒爽：开局有职业、有装备、有战斗
- 3 分钟留：想再刷一把 / 再抽一发 / 再冲一层

### 产品指标方向
- 次留靠每日运势 + 任务
- 中留靠 build / 套装 / 榜单
- 裂变靠同命理榜 + 分享卡

---

## 15. 当前结论

**“人生副本”下一阶段不该优先做网页 demo。**

应该优先钉死这个产品骨架：
- 命理做输入层
- 抽卡做爽点
- 刷关做中循环
- 装备做长线养成
- 排行榜做社交扩散

这才是能往微信小游戏和商业化方向走的结构。
