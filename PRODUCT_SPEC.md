# 人生副本 — Life RPG (Product Spec v0.1)

## 概念
把八字命理游戏化：用户输入生辰八字 → 生成一套 RPG 角色面板，包含职业、属性、技能、副本、Boss 等完整游戏体验。

## 输入
- 出生年、月、日、时辰（12 时辰选择器）
- 性别（可选）

## 核心算法流程
1. **四柱排盘**：公历 → 农历 → 天干地支四柱（年柱、月柱、日柱、时柱）
2. **五行计分**：按 METAPHYSICS_KNOWLEDGE.md 中的计分规则（天干 +5，地支藏干本气 +3/中气 +1/余气 +1）
3. **身强身弱**：月令分 + 根分 = S 值判定
4. **喜用神/忌神**：五行分数最低 1-2 个 = 喜用，最高 = 忌
5. **十神计算**：以日主为中心，判定其他七个干支的十神
6. **属性映射**：五行 + 十神 → RPG 六维属性

### 其他模块
- **等级** = 当前年龄
- **经验条** = 当前大运阶段进度
- **HP** = 五行平衡度 → 偏差越大 HP 越低
- **MP** = 印星强度
- **当前副本** = 流年运势描述（LLM 生成）
- **Boss 战** = 流年冲克分析（LLM 生成）
- **背包道具** = 喜用神对应颜色、方位、开运物
- **队友推荐** = 五行相生的职业

## 技术栈
- **前端**：HTML/CSS/JS（暗色 RPG 风格 UI）
- **八字引擎**：纯 JS 实现（万年历 + 天干地支 + 五行 + 十神计算）
- **可视化**：Canvas/SVG 雷达图 + HP/MP 条 + 技能树
- **叙事**：LLM API（副本描述、Boss 解说、角色故事）
- **分享**：html2canvas 生成分享卡片
